Propato Movil (Prototipo) Esta pequeña aplicación tiene como objetivo mostrar algunos controles y funcionalidades que posee el framework Ionic. Este framework nos permite desarrollar aplicaciones Hibridas, esto quiere decir que se puede ejecutar en cualquier dispositivo (smartphone, table, smartTv), en cualquier Sistema Operativo (Android,Ios,WindowsPhone) o en cualquier navegador.
La app esta conformada por distintos componentes:
Componente Descripción Login Pagina de autenticación de usuario Home Descripcion del prototipo (Usted esta aquí) Menú Lateral Contiene todas las pantallas accesibles desde el Home Ruta en curso En esta pantalla se visualiza el detalle de todos los destinos registrados para una ruta, puediendo acceder al detalle de cada remito y tambien al mapa donde indica la ubicación Rutas Asignadas En esta pantalla se visualiza el listado de todas las rutas que fueron asignadas al usuario Nueva Ruta Cuando se accede a esta opción se activara la camara del dispositivo para scanear una nueva ruta Salir Redirecciona la app al Login
